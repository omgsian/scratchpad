# About the project

This is a scratchpad application whereby the author can write a comment.

The comments are then saved to a MySQL database instance that runs on Azure.

# Tech stack

- Python v.3.10.5
- Flask v2.1.2
- npm
- tailwind@latest
- MySQL (Azure) - coming soon
- PythonAnywhere (host)
